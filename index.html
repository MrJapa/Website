OpenCV (Open Source Computer Vision Library) is a library of

programming functions mainly aimed at real-time computer vision.

Originally developed by Intel, it was later supported

by Willow Garage then Itseez (which was later acquired by Intel).

The library is cross-platform and free for use

under the open-source Apache 2 License.

Starting with 2011, OpenCV features GPU acceleration for real-time operations.
